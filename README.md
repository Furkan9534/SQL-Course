# SQLCOURSE
SQL Kursu
#sqlkursu

Bu depo, yeni başlayanlar ve junior geliştiriciler için pratik SQL örnekleri içermektedir.

Bu deponun amacı:
- SQL'i adım adım öğrenmek
- SQL'in gerçek dünya projelerinde nasıl kullanıldığını anlamak
- Arka uç ve veri odaklı uygulamalarda kullanılan yaygın sorguları uygulamak

## Konular

- Temel SELECT ve WHERE
- Filtreleme (LIKE, IN, BETWEEN, NULL)
- JOIN'ler
- GROUP BY ve HAVING
- Alt sorgular ve CTE
- Gerçek hayattan raporlama sorguları

## Nasıl kullanılır

Her klasör bir konuyu temsil eder.

Her `.sql` dosyası şunları içerir:
- amacın kısa bir açıklaması
- örnek sorgular

Sorguları şunlarda çalıştırabilirsiniz:

- SQL Server
- PostgreSQL
- MySQL (küçük sözdizimi farklılıkları olabilir)

 SQL Course
# sqlcourse

This repository contains practical SQL examples for beginners and junior developers.

The goal of this repository is:
- to learn SQL step by step
- to understand how SQL is used in real-world projects
- to practice common queries used in backend and data-driven applications

## Topics

- Basic SELECT & WHERE
- Filtering (LIKE, IN, BETWEEN, NULL)
- JOINs
- GROUP BY & HAVING
- Subqueries and CTE
- Real-life reporting queries

## How to use

Each folder represents one topic.
Each `.sql` file contains:
- a short description of the goal
- example queries

You can run the queries on:
- SQL Server
- PostgreSQL
- MySQL (minor syntax differences may apply)

## Who is this for?

- Beginners learning SQL
- Backend developers
- Data and AI oriented developers
